# 日历服务
> 日历和日程管理服务

### 目录结构说明

目录结构及用途说明：

```
calendar-service/src
└─main
  ├─java/com/lark/calendar
  │  └─com
  │      │  
  │      ├─config               配置类库
  │      ├─controller           前端控制类层
  │      ├─model                实体类
  │      ├─service              数据服务接口层
  │      │  └─impl              数据服务实现层
  │      └─utils                工具类库
  └─resources
          application.yml       项目配置
```
